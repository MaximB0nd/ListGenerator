Генератор списка случайных чисел

## Описание задачи
Сгенерировать список из 12 случайных целых чисел в диапазоне от 5 до 200 включительно.  
Параметры зависят от номера варианта (вариант №2):
- Номер по журналу: 2
- Количество элементов: 2 + 10 = 12
- Диапазон: от 5 до 2 × 100 = 200

## Среда разработки
- Редактор: PyCharm 
- Интерпретатор: Python 3.13
